## Использование

### Установка проекта:

```bash 
cd ./app
cp .env.local.example .env
```

Заполните `.env.local` файл своими значениями.

Необходимо поднять docker-контейнеры.

```bash
docker-compose up -d
```

### Запуск проекта:

```bash
docker-compose up -d
```

### Остановка проекта:

```bash
docker-compose down
```
